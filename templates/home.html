AgriNutrientNet - Deep Learning Based Estimation of Micronutrient Deficiency in crops

This website helps in identifying mineral nutrient deficiencies in crops using image analysis, AI, and deep learning. Users can upload a plant leaf image, and the system analyzes visual symptoms to detect possible nutrient deficiencies. It uses advanced machine learning models such as CNN for accurate prediction. The platform aims to provide a fast, cost-effective, and easy-to-use solution for improving crop health. This system supports smart and sustainable agricultural practices.
